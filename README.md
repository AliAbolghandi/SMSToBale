# SMS To Bale

اپلیکیشن اندرویدی که پیامک‌های یک شماره مشخص را عینا از طریق ربات بله برای یک مخاطب ارسال می‌کند.

## مرحله ۱: ساخت ربات در بله
۱. در اپلیکیشن بله، ربات **باتفادر (BotFather)** را پیدا کن.
۲. دستور `/newbot` را بزن و نام و یوزرنیم ربات را وارد کن.
۳. توکنی که برایت می‌فرستد را کپی کن (چیزی شبیه `123456:AAExxxxxxxxxxxxxxxxxxxxx`).

## مرحله ۲: ساخت اسکریپت گوگل (سرور واسط)
۱. به آدرس script.google.com برو و یک پروژه جدید بساز.
۲. محتوای فایل `apps-script/Code.gs` را کپی و در ویرایشگر جایگزین کن.
۳. مقدار `BALE_BOT_TOKEN` را با توکن ربات خودت جایگزین کن.
۴. مقدار `SECRET_KEY` را با یک رشته دلخواه و امن جایگزین کن (این عدد را جایی یادداشت کن، بعدا در اپ اندروید هم لازم است).
۵. از منوی **Deploy → New deployment** را بزن:
   - Type: **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
۶. آدرس Web App که چیزی شبیه `https://script.google.com/macros/s/xxxxx/exec` است را کپی کن.

## مرحله ۳: گرفتن Chat ID مخاطب
۱. مخاطب مقصد باید یک بار در اپ بله، ربات را پیدا کند و دستور `/start` را برایش بفرستد.
۲. در ویرایشگر اسکریپت گوگل، تابع `getChatIdHelper` را از نوار بالا انتخاب و اجرا (Run) کن.
۳. از منوی **View → Logs** خروجی را ببین؛ عدد `chat.id` همان Chat ID مخاطب است.

## مرحله ۴: ساخت APK با GitHub Actions (بدون نیاز به کامپیوتر قوی)
۱. یک حساب رایگان در github.com بساز (اگر نداری).
۲. یک ریپازیتوری جدید بساز (مثلا با نام `sms-to-bale`)، می‌تواند Private باشد.
۳. تمام فایل‌های همین پوشه (که برایت آماده شده) را در آن ریپازیتوری آپلود کن:
   - از طریق مرورگر: در صفحه ریپازیتوری روی **Add file → Upload files** بزن و کل پوشه را بکش و رها کن.
۴. بعد از آپلود، به تب **Actions** ریپازیتوری برو.
۵. یک workflow با نام **Build APK** می‌بینی؛ روی آن کلیک کن و دکمه **Run workflow** را بزن.
۶. حدود ۲ تا ۴ دقیقه صبر کن تا اجرا با علامت سبز تمام شود.
۷. روی همان اجرا کلیک کن، پایین صفحه بخش **Artifacts** را ببین و فایل `SmsToBale-debug-apk` را دانلود کن (یک فایل zip حاوی APK).

## مرحله ۵: نصب روی گوشی
⚠️ **مهم (فقط همین یک‌بار):** چون از این نسخه به بعد یک کلید امضای ثابت برای build اضافه شده، اگر نسخه قبلی اپ روی گوشی نصب است، اول آن را **Uninstall** کن (تنظیمات پاک می‌شوند، دوباره واردشان می‌کنی). از این به بعد، APK های بعدی همگی با همین کلید امضا می‌شوند و می‌توانی مستقیم روی نسخه قبلی نصب/آپدیت کنی بدون Uninstall و بدون از دست دادن تنظیمات یا تاریخچه.

۱. فایل zip دانلود شده را از حالت فشرده خارج کن؛ فایل `app-debug.apk` داخلش است.
۲. آن را به گوشی اندرویدی منتقل کن (تلگرام به خودت، ایمیل، یا کابل).
۳. روی فایل APK بزن؛ اگر پیام «نصب از منابع ناشناس» آمد، اجازه بده.
۴. اپ را نصب و باز کن.

## مرحله ۶: تنظیم اپ روی گوشی
داخل اپ چهار فیلد پر کن:
- **منبع(های) پیامک**: می‌تواند شماره معمولی، نام/Sender ID متنی بانک (مثل «بانک ملت»)، یا ترکیبی از چند مورد با کاما جدا شده باشد. مثال: `3000175, بانک ملت, +989121234567`
  - برای پیامک‌های بانکی که با اسم می‌آیند (نه شماره)، دقیقا همان نامی که در پیامک به‌عنوان فرستنده نشان داده می‌شود را وارد کن.
- **Chat ID مخاطب(ها)**: می‌توانی چند Chat ID را با کاما جدا کنی تا پیام هم‌زمان برای همه آن‌ها ارسال شود. مثال: `123456789, 987654321`
- **کلید محرمانه**: هنگام تایپ به‌صورت نقطه نمایش داده می‌شود (فیلد مخفی).

بالای صفحه دو نشانگر دسترسی (دریافت پیامک / خواندن پیامک) وجود دارد؛ اگر ضربدر قرمز داشته باشند یعنی آن دسترسی داده نشده — با ضربه روی هرکدام مستقیم به صفحه دسترسی‌های اپ در تنظیمات گوشی منتقل می‌شوی.

می‌توانی با سواپ روی صفحه (کشیدن انگشت به چپ/راست) هم بین تب «تنظیمات» و «تاریخچه» جابه‌جا شوی، نیازی به لمس دقیق تب نیست.

## فونت کدک
به دلیل محدودیت کپی‌رایت، فایل فونت کدک داخل پروژه قرار داده نشده. برای فعال‌سازی آن، فایل `app/src/main/res/font/README.txt` را بخوان.
- **آدرس Web App**: همان آدرسی که در مرحله ۲ کپی کردی.
- **Chat ID مخاطب**: همان عددی که در مرحله ۳ پیدا کردی.
- **کلید محرمانه**: دقیقا همان مقداری که در `SECRET_KEY` اسکریپت گذاشتی.

سپس دکمه «ذخیره تنظیمات» را بزن و مطمئن شو دسترسی SMS را هم داده‌ای.

⚠️ **نکته مهم:** برای این‌که اپ در پس‌زمینه هم کار کند، از تنظیمات گوشی، بهینه‌سازی باتری (Battery Optimization) را برای این اپ غیرفعال کن، وگرنه ممکن است سیستم‌عامل اپ را در پس‌زمینه ببندد.

## تست
یک پیامک از شماره تنظیم‌شده به گوشی ارسال کن؛ باید عین همان متن در چت ربات بله برای مخاطب مقصد ارسال شود.
